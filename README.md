PSLOG(1) User Commands.
=====

**NAME**

pslog - report current logs path of a process.

**SYNOPSIS**

pslog [options] pid [...]

**OPTIONS**

-V, --version
Output version information and exit.

**SEE ALSO**

ps(1), pgrep(1), pwdx(1)
